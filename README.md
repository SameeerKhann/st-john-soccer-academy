# St. John Soccer Academy — The Soaring Eagles

Website for St. John Soccer Academy, a youth soccer academy in Edmonton, AB developing boys and girls ages 4–17.

## Pages

- `index.html` — home: arc-reveal intro, cinematic video hero with scroll-driven programs panel, stats, testimonial marquee, gallery
- `programs.html` — programs & dark-gradient pricing tiers
- `about.html` — mission, values, Coach Johannes
- `code-of-conduct.html` — Parent Agreement & Code of Conduct (incl. the 24-Hour Rule)
- `faq.html` — 16-question FAQ
- `register.html` — registration form (opens a pre-filled email) + contact panel

## Stack

Static HTML/CSS/JS — no build step. GSAP (CDN) powers the homepage hero. Fonts from Google Fonts; photos/video hotlinked from Unsplash/Pexels.

To preview locally:

```
python -m http.server 8123
```

then open http://localhost:8123
